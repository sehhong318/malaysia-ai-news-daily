# Malaysia & AI Daily News

A public, source-linked news digest updated twice daily:

- **Malaysia News** — major Malaysian policy, economy, society, technology, and national developments.
- **AI News** — meaningful artificial-intelligence research, products, policy, safety, and industry developments.

## Archive structure

Each Malaysia date (`Asia/Kuala_Lumpur`, UTC+8) has one file:

```text
YYYY/
  MM/
    DD.md
```

Example: `2026/07/22.md`.

Each date file preserves two editions:

```text
10:00 MYT Edition
18:00 MYT Edition
```

The evening edition focuses on material developments since the morning edition and does not overwrite it.

## Editorial rules

- Prefer original sources such as official announcements, regulators, company/research blogs, papers, and government releases.
- Use reputable reporting when an original source is unavailable or additional context is necessary.
- Every item includes a direct link and publication date.
- Separate confirmed facts from interpretation; never invent missing details.
- Deduplicate syndicated or repeated stories within and across editions.
- Keep Malaysia and AI as separate sections.
- If no credible item is found for a section, state that explicitly rather than filling space.
